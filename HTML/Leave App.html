The IOM World Leave Management System is an internal workforce solution developed for IOM World Sri Lanka to streamline employee leave applications, overtime (OT) tracking, and attendance management. The project focused on delivering a smart, secure, and user-friendly system by integrating AI-powered geo-based face recognition and automated approval workflows.

Challenge

Managing employee leave, overtime, and attendance manually created inefficiencies, delays, and data inconsistencies across teams. The main challenge was to design a system that allows employees to apply for leave and OT seamlessly while ensuring accuracy, security, and compliance with organizational policies.

Additional constraints included integrating geo-based face recognition for attendance verification, handling different leave types and approval hierarchies, and ensuring the system remained easy to use for non-technical users. Time sensitivity, data privacy, and real-time validation were key factors that influenced design and technical decisions. To address these challenges, research was conducted on enterprise HR systems, AI-based attendance solutions, and user pain points in existing manual workflows. This helped prioritize features that reduced administrative effort while improving transparency and trust.

Solution

The final solution is a smart leave and workforce management platform that allows employees to apply for leave and overtime with minimal friction while providing managers with clear approval controls. I worked closely with my team to design and implement a system that integrates AI-powered geo-location and face recognition to validate attendance securely and accurately.

I was responsible for the complete UI/UX design, ensuring intuitive user flows for leave requests, OT submissions, approval tracking, and notifications. In addition to design, I actively contributed to the development process, implementing front-end components and aligning the system with the selected theme for a consistent and professional user experience. By combining intelligent automation, secure AI integration, and a clean UI, the system significantly improves operational efficiency, reduces manual errors, and delivers a modern HR experience tailored for IOM World Sri Lanka.
